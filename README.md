## 💻 Instalação e Execução

1. Clone o repositório: `git clone https://github.com/iaugustoz/nearby-app`

2. Acesse a pasta do projeto:
   `cd nearby-app`

3. Execute com Gradle (para backend):
   `./gradlew run`

4. Ou abra no Android Studio (para apps Android)
